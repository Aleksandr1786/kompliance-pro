# КомплаенсПро — Инструкция по установке

## Установка

1. Скачай `КомплаенсПро-Setup.exe` по ссылке, которую тебе прислали
2. Запусти установщик, дождись завершения установки
3. Приложение запустится автоматически и создаст ярлык на рабочем столе

## Последующие запуски

Просто открой ярлык «КомплаенсПро» на рабочем столе или в меню Пуск.

## Обновления

Приложение обновляется автоматически — при выходе новой версии ты увидишь
уведомление внутри самого приложения с предложением скачать и установить
обновление. Вручную ничего делать не нужно.

## Резервные копии

Не нужно искать файл базы данных вручную — в приложении уже есть встроенное
резервное копирование:

- **Настройки → Резервные копии → «Сделать бэкап сейчас»** — резервная копия
  создаётся сразу
- Автоматические резервные копии создаются приложением самостоятельно по
  расписанию (без твоего участия)

Если всё же нужно найти файл базы данных вручную (например, для переноса на
другой компьютер) — он называется `kompliance.json` и находится в папке
данных приложения (`%APPDATA%\КомплаенсПро\`, точное имя папки можно
уточнить в проводнике Windows: адресная строка → вставь `%APPDATA%` →
Enter — там будет папка с названием приложения).

## Если что-то пошло не так

Напиши в чат — разберёмся вместе.
