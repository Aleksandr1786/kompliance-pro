# КомплаенсПро — Инструкция по установке

## Первый запуск (один раз)

1. Распакуй папку `КомплаенсПро` на Рабочий стол

2. Открой Node.js (ярлык на рабочем столе или через поиск)

3. Введи команды по одной, нажимая Enter:

```
cd %USERPROFILE%\Desktop\КомплаенсПро
npm install
npm start
```

Установка займёт 2-3 минуты (скачиваются зависимости).
После этого откроется окно приложения.

## Последующие запуски

Каждый раз когда нужно открыть приложение:

```
cd %USERPROFILE%\Desktop\КомплаенсПро
npm start
```

> В следующей версии будет ярлык .exe — двойной клик и готово.

## Где хранятся данные

База данных находится по пути:
`C:\Users\[ИМЯ_ПОЛЬЗОВАТЕЛЯ]\AppData\Roaming\kompliance-pro\kompliance.db`

Это и есть файл который нужно копировать в резервную копию.

## Если что-то пошло не так

Напиши в чат — разберёмся вместе.
